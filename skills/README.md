# Skills

Публичные скиллы, которыми я делюсь.

Каждый скилл лежит в своей подпапке. Внутри — файлы скилла и краткое описание: что делает и как подключить.

## Структура

```
skills/
└── <название-скилла>/
    ├── SKILL.md        # описание и инструкции
    └── ...             # файлы скилла
```

## Как забрать скилл

- Скачай весь репозиторий архивом и скопируй нужную папку, **или**
- Склонируй репозиторий и забери конкретную папку скилла.

> Пока раздел пустой — скоро здесь появятся первые скиллы.
