# Skills

Публичные скиллы, которыми я делюсь.

Каждый скилл лежит в своей подпапке. Внутри — файлы скилла и краткое описание: что делает и как подключить.

## Каталог

| Скилл | Что делает |
|-------|------------|
| [meta-prompt/](meta-prompt/) | Превращает сырую идею в готовый, грамотно структурированный промпт с низким риском галлюцинаций |
| [reels-script/](reels-script/) | Превращает идею в вирусный сценарий Reels/Shorts: хук 1–3 сек, одна мысль, удар по болям |
| [carousel-script/](carousel-script/) | Превращает идею в карусель для Instagram/LinkedIn: обложка-хук, одна мысль/слайд, per-slide спек под рендер |

## Структура

```
skills/
└── <название-скилла>/
    ├── SKILL.md        # описание и инструкции
    └── ...             # файлы скилла
```

## Как забрать скилл

- Скачай весь репозиторий архивом и скопируй нужную папку, **или**
- Склонируй репозиторий и забери конкретную папку скилла.
