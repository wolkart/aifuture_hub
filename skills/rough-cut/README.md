# rough-cut — черновой монтаж записей

Скилл-«монтажёр черновой сборки»: синхронизирует отдельно записанный звук (OBS) с видео камеры по звуковой волне, вырезает тишину, филлеры («э-э», «пык-мык»), заикания и запоротые дубли (по транскрипту), а в скринкастах смотрит на кадры тихих участков и решает — вырезать, оставить или ускорить таймлапсом. Результат — cut-list JSON (список решений с причинами), из которого экспортируется таймлайн FCPXML для DaVinci Resolve (правишь резы руками перед рендером) и/или готовый mp4 для CapCut/рилсов/веба. Последнее слово всегда за тобой: спорные решения помечены, правка = одна фраза («в 02:15 верни») без пересчёта анализа.

## Требования

- **ffmpeg/ffprobe** — обязательно (macOS: `brew install ffmpeg`).
- **uv** — менеджер Python-зависимостей, скрипты самодостаточны через него (macOS: `brew install uv`).
- **Whisper** (для дублей/филлеров): ставится сам через uv при первом запуске — `mlx-whisper` на Apple Silicon, `faster-whisper` на остальных платформах. Модель скачивается при первом транскрипте (~1.5 ГБ, разово). Без whisper доступен режим «только тишина».

Всё работает локально — записи никуда не загружаются. При первом вызове скилл сам проверяет окружение и показывает, чего не хватает (ставит только с твоего согласия).

## Как пользоваться

- «Обработай запись: видео `~/rec/cam.mp4`, звук `~/rec/obs.wav`» — говорящая голова, синхрон + чистка (лёгкий профиль, автоматически).
- «Вот скринкаст `~/rec/screen.mp4`, места с загрузкой — таймлапсом» — полный профиль: решения по кадрам.
- «Порежь тишину, ничего не ускоряй» — лёгкий профиль явно.
- После отчёта: «в 02:15 верни, остальное ок» → правка cut-list → переэкспорт.

## Сценарии

| Вход | Что делает |
|---|---|
| Камера + звук OBS | Синхрон по волне → чистка по звуку/транскрипту |
| Скринкаст OBS (один файл) | Чистка + (полный профиль) таймлапс-решения по кадрам |
| Скринкаст + камера + звук | Мультикам: одни резы на обе дорожки, в Resolve — две видеодорожки |

## Выход

- `<имя>.cutlist.json` — список решений (канон; человекочитаемый, правится руками);
- `<имя>.fcpxml` — таймлайн для DaVinci Resolve (мягкие резы, маркеры SPEEDUP);
- `<имя>.roughcut.mp4` — готовый файл (H.264+AAC) для CapCut/рилсов/веба.

Исходники никогда не перезаписываются.

## Под себя

Пороги тишины, паддинги, модели whisper — `references/thresholds.md`; правила решений (филлеры/дубли/скринкаст) — `references/decide-rules.md`.
