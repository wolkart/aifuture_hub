# Пороги и параметры

Дефолты выбраны под разговорную запись с приличным микрофоном. Все крутятся флагами CLI.

## Тишина (`detect_silence.py`)

| Параметр | Дефолт | Когда крутить |
|---|---|---|
| `--noise-db` | −35 dB | Шумный фон (улица, вентилятор) → −30; студийная тишина → −40 |
| `--min-dur` | 0.8 с | Быстрая плотная речь → 0.5–0.6; медленная подача → 1.0 |
| `--pad` | 0.15 с | Паддинг внутрь интервала: сколько «дыхания» оставить вокруг речи. Речь обрезается на стыках → увеличить до 0.2–0.25 |
| `--min-cut` | 0.4 с | Резы короче этого не делаем (дёрганый монтаж хуже паузы) |

## Синхрон (`sync_offset.py`)

- `confidence < 0.2` — смещению не верить: разные помещения/сильный шум на камере. Просить хлопок в начале записи или ручное смещение.
- `--max-seconds 600` — корреляция по первым 10 минутам; для очень длинных записей достаточно.

## Кадры скринкаста (`extract_frames.py`)

- `edge = 0.2 с` — отступ от границ тихого участка (на границах бывает смаз перехода).
- Участок ≤ 0.4 с → один кадр из середины, длиннее → 3 кадра (начало/середина/конец).

## Транскрипт (`transcribe.py`)

- Модели по умолчанию: `mlx-community/whisper-large-v3-turbo` (Apple Silicon) / `large-v3` (faster-whisper). Слабое железо → `--model small` (хуже пословные таймкоды, но быстрее).
