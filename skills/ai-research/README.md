# ai-research — где применять

Скилл ресёрчит AI-мир из авторитетных источников и возвращает готовый артефакт: пул идей для контента, новостной дайджест или недельную подборку главных событий со ссылками. Результат — структурированный текст, не отчёт о размышлениях.

## Когда срабатывает

- **Идеи для контента про ИИ** — «о чём снять рилс», «дай темы для постов про AI», «контент-план на неделю».
- **Дайджест AI-новостей** — «собери дайджест», «топ-5 новостей за день», «что произошло в ИИ».
- **Подборка за неделю** — «подборка главных AI-новостей за неделю», «новости для клуба», «еженедельный обзор Пн–Вс».

НЕ для общих вопросов про ИИ и не для написания готового контента — скилл выдаёт сырьё, а не финальный пост/рилс.

## Три режима

`ideas` (дефолт) · `digest` · `weekly` — шаблоны полей и примеры заполнения в [references/output-modes.md](references/output-modes.md).

## Source-лесенка

Источник разрешается по приоритету:

1. **Perplexity MCP** — если подключён, вызывается первым; его ссылки идут как первичные источники.
2. **WebSearch / WebFetch** — если Perplexity недоступен или вернул мало результатов.
3. **Evergreen-генерация** — только если живой поиск полностью недоступен; явно помечается предупреждением «Живой поиск недоступен».

Домены по тирам (Tier-1 медиа, Tier-2 блоги лабораторий, Tier-3/4 агрегаторы, Tier-5 RU) — в [references/sources.md](references/sources.md).

### Подключить Perplexity MCP

```
cp .env.template .env
# открой .env и заполни PERPLEXITY_API_KEY=<твой ключ>
```

## Хэндофф

- **weekly** → `carousel-script` (каждый день = один слайд, тема «Неделя в AI», 7 слайдов).
- **ideas** → `reels-script` (идея как тезис) или `carousel-script` (идея как тема карусели).

## Примеры вызова

```
о чём снять рилс про AI на этой неделе
```

```
собери дайджест топ-5 AI-новостей за сегодня
```

```
подборка главных AI-событий за неделю для клуба
```

В ответ — структурированный артефакт по шаблону выбранного режима с реальными ссылками на источники.
