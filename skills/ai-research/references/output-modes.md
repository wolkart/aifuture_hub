# Форматы выхода режимов ai-research

Три режима скилла производят структурированный выход, готовый к дальнейшему использованию — посту, рилсу, карусели или передаче в другой скилл.

---

## Режим `ideas` — пул идей для контента

**Назначение:** генерация ранжированного списка тем для постов/рилсов/каруселей на основе свежих AI-новостей.

**Поля каждого кандидата:**

```
N. Тезис
   · Угол: <точка зрения или нарратив>
   · Почему зайдёт: <боль аудитории / тренд / провокация>
   · Формат: рилс | карусель | пост | обе
   · Источник: <название — URL>
```

**Пример (2 идеи):**

```
1. GPT-5 умеет рассуждать медленно — и это меняет всё
   · Угол: объясняю, чем «медленное мышление» отличается от обычного ответа
   · Почему зайдёт: создатели контента не понимают разницу и упускают фичу
   · Формат: карусель
   · Источник: OpenAI — openai.com/news/

2. Open-source обогнал GPT-4 по бенчмаркам: что это значит
   · Угол: провокация — нужны ли платные API в 2025?
   · Почему зайдёт: боль подписной платы + тренд на локальные модели
   · Формат: рилс
   · Источник: Papers with Code — paperswithcode.com
```

## Режим `digest` — дайджест топ-N новостей

**Назначение:** быстрый обзор самого важного за период (день / неделя), готовый к публикации или пересылке.

**Поля каждой новости:**

```
N. Заголовок
   Суть: <1–2 строки — что случилось и зачем это важно>
   Ссылка: <URL>
   Почему важно: <контекст, влияние, масштаб>
```

**Пример (2 новости):**

```
1. Anthropic выпустила Claude 4 с расширенным контекстным окном до 500K токенов
   Суть: новая модель обрабатывает книги целиком за один запрос — без разбивки на чанки.
   Ссылка: anthropic.com/news/claude-4
   Почему важно: меняет архитектуру RAG-систем; компании смогут отказаться от векторных БД в ряде сценариев.

2. Meta открыла веса Llama 4 Scout с коммерческой лицензией
   Суть: 17B-параметровая модель с MoE-архитектурой доступна для коммерческого использования бесплатно.
   Ссылка: ai.meta.com/blog/llama-4-scout
   Почему важно: снижает порог входа в локальный AI для стартапов; прямой удар по платным API.
```

## Режим `weekly` — структура недели (хэндофф в carousel-script)

**Назначение:** семь строк (Пн–Вс) с главной новостью каждого дня, готовых к передаче в скилл `carousel-script`.

**Важно: это хэндофф-структура для `carousel-script`** — каждая строка = один слайд карусели. Передавай весь блок как контент для скилла `carousel-script`, указав тему «Неделя в AI» и формат 7 слайдов.

**Поля каждой строки:**

```
День · Главная новость · 1 строка сути · Ссылка
```

**Пример (2 дня из 7):**

```
Пн · OpenAI анонсировала o3-mini для разработчиков · Лёгкая версия reasoning-модели доступна через API с 1 февраля · openai.com/news/o3-mini

Вт · Google DeepMind опубликовала AlphaFold 3 для некоммерческих задач · Предсказание структур белков и молекул — теперь бесплатно для науки · deepmind.google/discover/blog/alphafold3
```

**Полный шаблон недели (для заполнения):**

```
Пн · [новость] · [суть] · [ссылка]
Вт · [новость] · [суть] · [ссылка]
Ср · [новость] · [суть] · [ссылка]
Чт · [новость] · [суть] · [ссылка]
Пт · [новость] · [суть] · [ссылка]
Сб · [новость] · [суть] · [ссылка]
Вс · [новость] · [суть] · [ссылка]
```
