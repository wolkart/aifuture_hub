# strategist — где применять

Жёсткий стратег-коуч по позиционированию: не поддакивающий ассистент, а хирург, который вскрывает то, что ты сам не видишь. Методология — позиционирование (Траут/Райс), категорийный дизайн, JTBD, Value Proposition Canvas, Хормози.

## Два режима

- **ИНТЕРВЬЮ** — вытащить позиционирование с нуля: «проведи стратегическое интервью», «собери мою стратегическую карту», «выведи УТП/аватар/миссию». Ведёт по 4 фазам (Рентген → Вскрытие продукта → Позиционирование → Сборка), один вопрос за раз, не принимает первый ответ, в конце собирает стратегическую карту (6 разделов) как артефакт. Ты сам сохраняешь результат — скилл ничего не перезатирает.
- **ДРЕЙФ-ЧЕК** — проверить готовый материал против позиционирования: «проверь этот пост/CTA/оффер на дрейф», «где я сплыл в абстракцию». Быстрый разбор по рубрике (аватар · враг · обещание · конкретность · продажа vs ценность · CTA) → «ДРЕЙФ / ВОЗВРАТ / ВЕРДИКТ».

## Что даёт / чего не делает

- Даёт: жёсткое интервью → стратегическую карту; хирургический дрейф-чек контента/решений.
- НЕ пишет промпты (это `meta-prompt`), НЕ пишет готовый пост/сценарий (это `post-hq` / `reels-script` / `carousel-script`). Он про стратегию, не про финальный текст.

## Генерик + твоё позиционирование

Скилл обезличенный (работает с любым брендом). Твоё позиционирование — это **вход**: для дрейф-чека дай свою карту/CTA/аватар или укажи файл (например, из волта/`content-studio`). Метод интервью — проверенное ядро, менять его не нужно.

## Пример вызова

```
проведи со мной стратегическое интервью — хочу вытащить позиционирование
```
```
проверь этот CTA на дрейф от моего позиционирования: "..."  (аватар/карта — вот)
```
