# post-hq — штаб постов

Скилл ведёт идею через четыре слоя до готового поста-артефакта: **голос** (brand-voice) → **цель** (контентный или продающий) → **площадка** (Telegram, Threads/X, LinkedIn, Instagram) → **ревью** до вердикта 🟢. Работает как оркестратор: диспетчеризует задачу, читает нужные модули по очереди и передаёт карточки скиллу `carousel-script`.

## Что делает

- **Лестница вмешательства** — оценивает вход (сырьё / полуфабрикат / почти готово) и выбирает степень касания. Чем готовее вход, тем меньше переписывает.
- **Роутинг платформа × цель** — если координаты не указаны, рекомендует сам с обоснованием. Если из входа нельзя надёжно определить — задаёт ровно один уточняющий вопрос.
- **Условная загрузка фреймворков** — `selling`: Хормози (Value Equation, Grand Slam, усилители, гарантии, scarcity/urgency) + Огилви (заголовок, body-copy); `informational`: только Огилви. Хормози к экспертным постам не подмешивается.
- **Порядок слоёв (строгий):** каркас по цели → `writing-principles` → голос последним. Голос не латает смысловые дыры — он одевает готовый каркас и срезает инфоцыганщину (§9 brand-voice).
- **Review-цикл** — после генерации запускает роль-ревьюер: светофор §6, анти-шаблон §4, вёрстка §8, чек-лист платформы, честность оффера. Вердикт 🔴/🟡 → конкретные правки → снова писателю → повторить до 🟢.
- **Делегирование карточек** — если задача включает карусель (Threads, Instagram) — визуал делегируется `carousel-script`, текст поста и карточки — параллельные артефакты, не дубли.

## Когда срабатывает

«Напиши пост», «напиши пост в телегу», «продающий пост», «прогрев», «оформи в моём голосе», «оформи в моём стиле» — или когда пользователь присылает сырьё, тезисы, наговорённый каркас / черновик и просит сделать из этого пост.

Также вызывается из `content-machine` как формат «пост»: оркестратор передаёт сюда суть идеи (и опц. площадку) — штаб сам разруливает площадку × цель × голос × review.

## Что внутри

| Файл | Содержимое |
|------|-----------|
| `SKILL.md` | Оркестратор: Шаг 0, лестница входов, роутинг, условная загрузка, порядок слоёв, review-цикл, делегирование, границы |
| `references/platforms/telegram.md` | Модуль площадки: лимиты, структура, вёрстка (ссылка на brand-voice §8), типы CTA, чек-лист, антипаттерны |
| `references/platforms/threads-x.md` | Модуль площадки: короткий формат (X ≤280 / Threads ≤500 символов), одиночный пост + нить (тред), типы CTA, чек-лист, антипаттерны |
| `references/platforms/linkedin.md` | Модуль площадки: профессиональный регистр (голос сохранён), сильные первые строки (до «see more»), хэштеги, чек-лист, антипаттерны |
| `references/platforms/instagram-caption.md` | Модуль площадки: подпись-компаньон визуала (пост/карусель/рилс), хук до обрезки (~125 симв.), хэштеги-блок, связка с carousel-script |
| `references/purposes/informational.md` | Роль «экспертный/контентный пост»: полезное действие, миф→разбор, числа-якоря, личный опыт + Огилви |
| `references/purposes/selling.md` | Роль «продающий/прогревочный пост»: скелет убеждения по Хормози + Огилви, страховка §9 против инфоцыганщины |
| `references/frameworks/hormozi.md` | Дистиллят приёмов продаж по Хормози: Value Equation, Grand Slam Offer, усилители, гарантии, scarcity/urgency |
| `references/frameworks/ogilvy.md` | Дистиллят приёмов Огилви: большая идея, правила заголовка, body-copy, ясность > оригинальности |
| `references/review.md` | Роль-ревьюер: светофор §6, анти-шаблон §4, вёрстка §8, чек-лист платформы, честность оффера; при браке — конкретные замечания писателю |

Голос не дублируется внутри штаба — читается из `skills/brand-voice/references/` как единый источник правды.

## Что НЕ делает

Скилл не постит и не доставляет контент — это задача слоя рантайма. Не переписывает авторский замысел: при конфликте побеждает автор. Не делает визуал карточек — делегирует `carousel-script`.

## Фазы

**Фаза 1 ✅:** Telegram (контентный + продающий пост).

**Фаза 2 ✅:** Threads/X, LinkedIn, Instagram-подписи — активны. Модули площадок по единому контракту платформы: `references/platforms/threads-x.md`, `linkedin.md`, `instagram-caption.md`.
