# carousel-script — где применять

Скилл превращает идею или тезис в готовую карусель для Instagram и/или LinkedIn: цепляющая обложка, одна ключевая мысль, раскладка по слайдам с атомизацией, плюс per-slide спек для рендера. Результат — артефакт карусели, а не объяснение темы.

## Когда срабатывает

- **Карусель / пост-карусель** — кидаешь идею со словами «сделай карусель», «карусель для инсты/линкедина», «пост в слайдах», «придумай карусель» → получаешь готовую карусель с per-slide спеком.
- **Кросс-перенос формата** — берёшь чужую залетевшую карусель из любой ниши и переносишь приём/угол на свою тему. Меняются примеры и угол, шаблонность помечается.

НЕ для коротких видео (это reels-script), длинных видео или обычных текстовых постов.

## Две платформы

**IG** (живой тон, провокация, 6–10 слайдов) и **LinkedIn** (профессиональный инсайт, 8–12 слайдов) — платформенные брифы в [references/instagram.md](references/instagram.md) и [references/linkedin.md](references/linkedin.md). Если платформа не указана, скилл генерирует две отдельные версии.

## Per-slide спек

На каждый слайд скилл выдаёт структурированный спек: роль слайда, заголовок, body-текст, layout-тип, визуальная пометка (акцент / иконка / иллюстрация). Полная схема полей — [references/slide-spec.md](references/slide-spec.md).

## Рендер-лесенка

Спек выдаётся всегда. Рендер — opt-in:

1. **Ручной режим (приоритетный override, по умолчанию)** — берёшь спек и собираешь слайды руками в Figma, Canva или любом редакторе. Это дефолт, если рендер не запрошен явно.
2. **Авто-рендер через Figma MCP** — подключается по запросу; скилл передаёт спек агенту рендера через Figma API.
3. **Фоллбэк image-gen** — если Figma MCP недоступен, скилл использует image-gen API для генерации картинок по спеку.

Чтобы подключить авто-рендер: скопировать `.env.template` → `.env`, заполнить `FIGMA_API_TOKEN` и `FIGMA_FILE_KEY` (и/или `IMAGE_API_KEY`).

## Пример вызова

```
сделай карусель про то что большинство людей используют ChatGPT неправильно
```

```
переделай эту LinkedIn-карусель про тайм-менеджмент под нишу ИИ-инструментов
```

В ответ — карусель по слайдам (обложка + тело + слайд-вывод + CTA), per-slide спек, метастрока угла и блок «Допущения».
