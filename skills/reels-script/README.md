# reels-script — где применять

Скилл превращает сырую идею или тезис в готовый вирусный сценарий Reels/Shorts: хук 1–3 сек, одна ключевая мысль, широкая подача без техножаргона. Результат — текст сценария как артефакт, не отчёт о размышлениях.

## Когда срабатывает

- **Сценарий короткого видео** — кидаешь идею со словами «сними про…», «сделай рилс», «сценарий для шортс», «придумай вирусное видео» → получаешь сценарий по битам.
- **Свой тезис или хук** — даёшь готовую первую фразу или мысль, скилл собирает вокруг неё ролик (хук сохраняет или отвергает с конкретной причиной, не подменяет молча).
- **Переупаковка чужого вирусного ролика под свою нишу** — берёшь залетевший рилс из любой темы (фитнес, финансы) и переносишь приём/угол на свою (по умолчанию ИИ). Не копия: меняются примеры и угол, кросс-нишевость помечается.

НЕ для длинных видео, постов или каруселей.

## Две оси формата

Сценарий описывается как **угол** (что: инструкция · разоблачение мифа · провокация · кейс-история · подборка) × **манера** (как снято: разговорный · новостной · личная история) — детали в [references/formats.md](references/formats.md).

## База хуков

Источник хука разрешается по приоритету (см. «Порядок разрешения источника хука» в [SKILL.md](SKILL.md)): **хук пользователя → внешняя база паттернов → встроенный хребет**. Внешняя база подключена скиллом [hook-base](../hook-base/): на шаге 2 reels-script вызывает его (операция ВЗЯТЬ) и берёт адаптированные под тему хуки; пусто по теме — падает на встроенный хребет.

## Пример вызова

```
сделай рилс про то что нейросети помогают экономить время
```

```
переупакуй вот этот вирусный фитнес-рилс с хуком "ты тренируешься годами
и не растёшь — вот почему" под нишу ИИ
```

В ответ — сценарий по 5 битам (хук · усиление · контент · вывод · лидмагнит), строка «почему хук цепляет», метастрока паттерна и блок «Допущения».
