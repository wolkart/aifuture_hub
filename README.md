# AI Future Hub

Хаб полезных материалов, которыми удобно делиться. Каждая папка — отдельный раздел: скиллы, автоматизации, шаблоны и другие классные штуки.

Любой материал можно **скачать архивом** (кнопка `Code → Download ZIP`) или **склонировать** репозиторий целиком:

```bash
git clone https://github.com/wolkart/aifuture_hub.git
```

## Разделы

| Раздел | Что внутри |
|--------|------------|
| [skills/](skills/) | Публичные скиллы, которыми я делюсь с подписчиками |

> Список будет пополняться: автоматизации (n8n), шаблоны, и другие материалы.

## Как пользоваться

1. Зайди в нужную папку — внутри будет свой `README.md` с описанием.
2. Скачай весь репозиторий архивом или склонируй, либо забери только нужную папку.
3. Следуй инструкциям внутри конкретного материала.

## Как подключить скиллы к Claude Code

Папка `skills/` — это **дистрибутив**: чтобы скилл срабатывал в Claude Code, его нужно положить туда, где Claude ищет скиллы. Удобнее всего — **ссылкой** на склонированный репозиторий (правишь/обновляешь репо — скилл сразу актуален), не копируя файлы.

### Все скиллы сразу (рекомендую)

Скрипты идемпотентны — подключают новые скиллы и не трогают уже подключённые. Запускай повторно после `git pull`, когда в репо появятся новые скиллы.

```powershell
# Windows (без прав админа)
git clone https://github.com/wolkart/aifuture_hub.git
powershell -ExecutionPolicy Bypass -File .\aifuture_hub\skills\install-skills.ps1
```

```bash
# macOS / Linux / Git Bash
git clone https://github.com/wolkart/aifuture_hub.git
bash ./aifuture_hub/skills/install-skills.sh
```

### Один скилл вручную

```powershell
# Windows — junction (mklink /J): работает без админа и Developer Mode
cmd /c mklink /J "$env:USERPROFILE\.claude\skills\reels-script" "C:\путь\к\aifuture_hub\skills\reels-script"
```

```bash
# Linux / macOS / Git Bash
ln -s "$(pwd)/aifuture_hub/skills/reels-script" ~/.claude/skills/reels-script
```

После этого скилл доступен **во всех проектах**. Для одного проекта вместо `~/.claude/skills/` используй `.claude/skills/` внутри него.

> **Перенёс репозиторий?** Ссылки хранят абсолютный путь и станут битыми. Удали их из `~/.claude/skills/` (`Remove-Item <ссылка>` / `rm <ссылка>` — файлы скилла в репо это не трогает) и запусти скрипт установки заново.

> Установку «в один клик» (через `/plugin`) добавлю позже, когда оформлю хаб как плагин-маркетплейс.
