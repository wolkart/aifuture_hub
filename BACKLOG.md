### Конструкции тела (сид для базы подходов — выжимка из библиотеки 3 авторов)

Сходятся у всех трёх авторов (Ник Сараев / Косолапов / солянка). Это скелеты ТЕЛА ролика под showcase/новость, НЕ хуки:

- **Анонс + халява + коммент-бейт:** «{кто} только что выкатил {что}» → 3–4 буллета возможностей → «и главное — бесплатно / open-source / без лимитов» → коммент-бейт «напиши кодовое слово — кину ссылку».
- **Отрицание лидера → замена:** «забудь про {лидер} / больше не плати за {лидер}» → чем новый инструмент лучше → как запустить (онлайн или локально) → CTA.
- **Иностранный авторитет → пошаговый туториал:** «этот индус/американец показал мне X» → 3–5 шагов с экрана → «готово за N секунд» → «сохрани / скинь другу».
- **Стыд через вопрос / «большинство делает неправильно»** → правильный способ → персональный результат у зрителя.
- **Сравнение / «убийца»:** «X стоит N, а тут 0» или «рвёт {лидеры} по бенчмаркам» → перечисление превосходств → «бесплатно / open-source» → CTA.
- **Сценка-эксперимент:** разыгранный мини-скетч/диалог → к середине вскрывается, что всё сделал ИИ → CTA.
- **История провала→успеха:** «я облажался N раз / разорился на подписках» → перелом → решение-сервис → CTA.
- **Нумерованный разбор ошибок:** «вот как правильно: 1, 2, 3» с контрастом «не делай так — делай так».
- **Пусть читатель посчитает сам:** описываешь не продукт, а ЧЕЛОВЕКА с маленьким ежедневным действием («100 комментов в день» / «30 минут вечером») → растягиваешь во времени → задаёшь вопрос «сколько у него будет через год?» → **пауза, отдельное сообщение** → только потом payoff. Читатель считает в голове, и своей цифре верит абсолютно. Две ветки: считать **выгоду** (действие × время = большое число) или **свою цену** («30 мин × месяц = 15 часов; прикинь, сколько ты листаешь ленту»). Ломается, если: числа не компаундятся (соврёшь), нет отдельного сообщения (читается как инородное тело), отвечаешь за читателя вместо паузы, подпираешь выдуманной статистикой. Требует места — строй сообщение ВОКРУГ приёма, не вставляй в готовое.
